# 📩 Contact Form Website

A **modern and responsive Contact Form** built using **HTML, CSS, and JavaScript**.  
This project demonstrates how to create a clean user interface for collecting user messages through a web form.

🔗 **Live Demo**  
https://01contactform.vercel.app/

🔗 **GitHub Repository**  
https://github.com/amannpal98/Web-Design-And-Devlopment/tree/main/Contact_Form

---

## 🚀 Features

- Responsive contact form UI
- Clean and modern design
- User input fields for name, phone, email, and message
- Mobile-friendly layout
- Easy to integrate into any website
- Simple and lightweight front-end project

The contact form allows users to enter details such as **name, phone number, email, and feedback**, making it a typical example of a front-end contact form used on many websites. :contentReference[oaicite:0]{index=0}

## 🛠️ Technologies Used

- **HTML5** – Structure of the webpage
- **CSS3** – Styling and layout
- **JavaScript** – Form behavior and validation
- **Vercel** – Deployment and hosting
